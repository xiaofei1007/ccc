# BioPatch+ Demo

一个展示未来可穿戴健康监控设备的React演示应用。

## 功能特性

- 🏥 实时健康数据监控（血压、心率、血氧、血糖）
- 📊 24小时健康数据图表
- ⚠️ 智能健康预警系统
- 💉 自动化治疗建议
- 👤 用户注册和登录系统
- 📱 响应式设计

## 技术栈

- **Frontend**: React 18
- **UI库**: shadcn/ui + Radix UI
- **样式**: Tailwind CSS
- **图表**: Recharts
- **图标**: Lucide React

## 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm run build
```

## 演示流程

1. **注册账户** - 体验完整的注册流程
2. **健康监控** - 查看实时健康数据
3. **自动预警** - 15秒后触发血糖预警
4. **治疗决策** - 选择批准或拒绝自动治疗

## 注意事项

此应用仅用于演示目的，不提供真实的医疗建议或治疗。所有健康数据均为模拟生成。
